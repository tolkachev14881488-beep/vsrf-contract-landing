# GitHub Pages отключён

Автопубликация на GitHub Pages удалена из репозитория.

## Если сайт ещё открывается по github.io

1. Откройте: https://github.com/tolkachev14881488-beep/vsrf-contract-landing/settings/pages  
2. **Build and deployment → Source:** выберите **None** (или отключите Pages).  
3. Сохраните.

## Домен sluzhbarf.ru

Пока DNS указывает на IP GitHub Pages, сайт может открываться с их серверов даже без workflow.

Чтобы полностью убрать сайт с GitHub:
- отключите Pages в настройках (см. выше), **и/или**
- смените A-записи домена на IP вашего нового хостинга.

Файлы сайта для загрузки на хостинг: папка `delivery/sluzhbarf-full/сайт/` или архив `delivery/sluzhbarf-full.zip`.
